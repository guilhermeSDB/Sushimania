## 🚀 Instalando Sushimania

Para instalar o Sushimania, siga estas etapas:

```
git clone https://github.com/guilhermeSDB/Sushimania.git
```

## ☕ Usando Sushimania

Para usar Sushimania, apenas abra index.html
